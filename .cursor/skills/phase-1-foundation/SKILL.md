# Fase 1 — Foundation

## Instruções

1. Leia `orchestrators/PHASE_1_foundation.md` e execute todas as etapas em ordem.
2. Atue como `ARCHITECT_AGENT` — leia também `agents/ARCHITECT_AGENT.md`.
3. Use a stack definida em `templates/project_config.md`.
4. Siga `.cursor/rules/global.mdc` e `.cursor/rules/database.mdc` para schema Prisma.

## Critério de conclusão

Projeto roda localmente com `npm run dev`.
